ScrambleScorer UI Directions

3 design directions × 3 screens each — no build, no deploy. Click tabs to explore.

D — Augusta Modern: Masters palette (greens + gold) combined with C's clean white-card layout. Green gradient header, white cards on light gray, Georgia serif for branding, sans-serif for data. Hero leader card, grouped hole cards for score entry, modern rounded inputs.
A — Augusta Elevated: Keeps the Masters palette but adds depth — gradient header, richer gold accents, trophy podium for top 3, stronger score typography, subtle texture. Evolutionary, familiar feel.
B — Dark Scoreboard: Full dark mode inspired by PGA Tour Live and ESPN leaderboards. High contrast gold on near-black, glowing score highlights, broadcast-style layout. Bold departure from current design.
C — Clean & Modern: White-dominant, editorial feel with big bold score numbers as the hero. Subtle gray cards, no heavy color blocks. Score colors are still present but text-only. Upscale country club aesthetic.

★ D — What's different
  • Masters green gradient header (A) on white-card body (C)
  • Georgia serif for brand titles, sans-serif for data
  • Hero card for leader with green accent stripe
  • Gold double-circle for eagle, red circle for birdie
  • Empty holes show green-tinted + button as next action
  • Chat green header bar + circular green send button
  • Home has compact header + recent matches list
  • Official Scoring badge in gold
A — What's different
  • Gradient header instead of flat green
  • Gold circle badges for top 3 positions
  • Live dot indicator on leaderboard
  • Leader row has subtle gold background + description text
  • Chat uses rounded pill style
  • Home has serif style kept, card-in-card layout
  • Score row has Hole/Par labels above numbers
B — What's different
  • Full dark mode — near-black backgrounds
  • Leader score glows gold with box-shadow
  • Gold left-border accent on leader row
  • Birdie/eagle badges glow with matching color
  • Sans-serif font — more broadcast feel
  • Chat input is dark, gold send button
  • Home uses subtle gradient hero + gold CTA
C — What's different
  • White-dominant, gray background — very clean
  • Leader gets a hero card treatment, not just a row
  • Score badges are in individual cards per hole
  • Chat uses pill-shaped input + circular send button
  • Home has "Recent matches" section
  • Sans-serif throughout — modern app feel
  • Dashed border "Create" button — inviting
